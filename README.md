# pathfinding assignment

u can use a* or greedy bfs to find the path from start to goal

features:
- grid world
- start and goal are fixed
- random mazes (press button)
- click to make walls
- dynamic mode kinda works i think
- shows visited nodes and frontier and stuff

files:
node.py - node class
grid.py - the grid
search.py - the search algos (astar, greedy)
agent.py - the agent code
gui.py - pygame visualization
main.py - run this one

setup:
need pygame
pip install -r requirements.txt
python main.py
